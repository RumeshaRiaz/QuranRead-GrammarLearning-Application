# Noor ul Bayan – Quran Learning App

Noor ul Bayan is a modern Quranic learning application designed to help users read, understand, and connect with the Qur’an in a simple, calm, and progressive way.

## 🌙 Features

### 📖 Quran Reading Mode
- Clean and distraction-free Quran reading experience
- Arabic text by default
- Optional Urdu & English translations
- Tap on ayah to explore meaning and relationships
- Word-level interaction with meanings and grammar

### 📚 Arabic Grammar Learning
- Learn Quranic Arabic from basics
- Covers Ism, Fi‘l, Harf
- Sentence structures and word relationships
- Step-by-step learning approach
- Beginner-friendly and progressive

### 🧠 Smart Learning Experience
- No overwhelming information
- Grammar & details shown only when needed
- Visual learning instead of long text
- User-controlled depth of learning

### 📱 Modern UI/UX
- Dark theme with elegant Islamic aesthetics
- Minimal, clean, and premium design
- Smooth and intuitive navigation

### ⚡ Offline Support
- Core features work without internet
- No login required

## 🎯 هدف (Purpose)
To provide a peaceful and effective way to understand the Qur’an by combining reading with foundational Arabic grammar.

## 🚀 Future Improvements
- AI-based learning assistance
- Tajweed enhancements
- Progress tracking system
